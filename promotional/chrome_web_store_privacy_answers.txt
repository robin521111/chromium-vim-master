=== Chrome Web Store 隐私权规范填写内容 ===

【单一用途说明】
rVim 是一个专门为 Google Chrome 浏览器添加 Vim 风格键盘导航和操作的扩展。其单一用途是通过键盘快捷键提供高效的网页浏览体验，包括页面导航、链接跳转、文本搜索、标签页管理等功能，完全替代鼠标操作。

【远程代码使用理由】
rVim 不使用远程代码。所有功能代码都包含在扩展包中，确保用户隐私和安全。

【主机权限使用理由】
rVim 需要在所有网站上提供 Vim 风格的键盘导航功能，包括在任何网页上注入键盘快捷键处理脚本、提供链接提示和页面导航功能、实现文本搜索和选择功能，确保用户可以在任何网站上使用 Vim 操作方式。

【bookmarks 权限使用理由】
允许用户通过键盘快捷键快速访问和管理书签，包括使用 :bookmarks 命令快速搜索和打开书签、通过键盘快捷键添加当前页面到书签、提供书签的快速导航和管理功能。

【clipboardRead 权限使用理由】
支持从剪贴板粘贴内容的 Vim 操作，包括实现 Vim 风格的粘贴命令（如 p 键）、允许用户将剪贴板内容粘贴到搜索框或地址栏、提供与系统剪贴板的无缝集成。

【clipboardWrite 权限使用理由】
支持复制内容到剪贴板的 Vim 操作，包括实现 Vim 风格的复制命令（如 y 键复制链接或文本）、允许用户快速复制当前页面 URL、支持选中文本的快速复制功能。

【downloads.shelf 权限使用理由】
管理下载栏的显示和隐藏，通过键盘快捷键控制下载栏的显示状态，提供更清洁的浏览界面，支持 Vim 风格的界面管理。

【downloads 权限使用理由】
提供下载管理的键盘操作，包括通过键盘快捷键查看和管理下载项目、支持下载文件的快速访问、实现下载历史的键盘导航。

【history 权限使用理由】
提供浏览历史的键盘导航和搜索，包括使用 :history 命令快速搜索浏览历史、通过键盘快捷键在历史记录中导航、支持历史页面的快速访问和管理。

【sessions 权限使用理由】
管理浏览器会话和标签页状态，包括保存和恢复标签页会话、支持会话的键盘管理、提供标签页状态的持久化存储。

【storage 权限使用理由】
存储用户的 rVim 配置和设置，包括保存用户自定义的键盘映射和配置、存储扩展的设置和偏好、确保配置在浏览器重启后保持。

【tabs 权限使用理由】
提供标签页的完整键盘管理功能，包括通过键盘快捷键切换、创建、关闭标签页、支持标签页的重新排序和移动、实现标签页间的快速导航、提供标签页信息的访问。

【topSites 权限使用理由】
快速访问最常访问的网站，包括通过键盘快捷键打开常用网站、提供网站访问频率的快速导航、支持个性化的网站快速访问。

【webNavigation 权限使用理由】
监听页面导航事件以正确初始化 rVim 功能，包括确保在页面加载时正确注入 rVim 脚本、处理单页应用的导航变化、维护 rVim 功能在页面切换时的一致性。

【醒目披露（商品详情建议加入）】
rVim 仅为其单一用途处理必要数据，不收集个人身份信息，不集成分析或广告；仅在用户触发时执行打开页面、下载或从 GitHub Gist 加载配置等必要请求，不进行后台遥测或数据上报。命令与补全历史仅在用户同意并开启选项后才在本地保存，且可随时关闭与清除。支持仅本地存储与关闭远程配置（离线模式）。
