## 设计特色

### 视觉风格
- **配色方案**：深蓝色背景 (#2c3e50)，绿色文字 (#2ecc71)，灰色按键
- **设计理念**：现代简洁，突出 Vim 特色
- **品牌元素**：rVim logo 包含 "vim" 文字和 "hjkl" 导航键

### 功能展示
1. **链接提示 (Link Hints)**：按 'f' 键显示橙色提示标签
2. **命令模式 (Command Mode)**：按 ':' 键进入命令行界面
3. **Vim 导航 (Vim Navigation)**：使用 hjkl 键进行页面导航

## 文件格式

每个图片都提供两种格式：
- **SVG 格式**：矢量图，可无损缩放，适合编辑和定制
- **PNG 格式**：位图，适合直接使用和发布

## 使用建议

- Chrome Web Store：使用 screenshot_1280x800.png
- 社交媒体：使用 small_tile_440x280.png
- 网站横幅：使用 large_tile_1400x560.png
- 自定义尺寸：使用对应的 SVG 文件重新导出

## 技术规格

所有图片均符合以下标准：
- 高分辨率，适合高清显示
- 透明背景支持（SVG 格式）
- 现代浏览器兼容
- 文件大小优化
