rVim 搜索功能测试页面

测试说明：

  • 按 / 键开始搜索
  • 按 ? 键开始反向搜索
  • 按 n 键跳转到下一个搜索结果
  • 按 N 键跳转到上一个搜索结果
  • 按 Esc 键退出搜索模式

预期效果：搜索状态应该显示在右下角，格式为 "当前结果 / 总结果数"

问题检查清单

  1. 搜索状态是否显示在右下角？
  2. 搜索状态格式是否正确（如 "1 / 5"）？
  3. 用户输入的搜索内容是否在搜索框中正确显示？
  4. 搜索结果高亮是否正常？
  5. n/N 键导航是否正常工作？
  6. 搜索状态是否实时更新？
