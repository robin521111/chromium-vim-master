rVim Chrome 扩展功能测试页面

1. 基本功能测试

测试步骤：
  • 点击浏览器工具栏中的 rVim 图标
  • 检查弹出窗口是否正常显示
  • 测试 "Disable rVim" 和 "Enable rVim" 按钮
  • 测试 "Settings" 链接

2. 键盘导航测试

测试按键：
  • f - 在当前标签页打开链接提示
  • F - 在新标签页打开链接提示
  • j - 向下滚动
  • k - 向上滚动
  • h - 向左滚动
  • l - 向右滚动
  • gg - 滚动到页面顶部
  • G - 滚动到页面底部

测试链接（按 f 或 F 键激活链接提示）：

Google GitHub Stack Overflow Mozilla W3C 页面内链接1 页面内链接2 页面内链接3

3. 命令栏测试

测试步骤：
  • 按 : 键打开命令栏
  • 尝试输入命令如 :help
  • 测试命令自动补全功能
  • 按 Esc 键关闭命令栏

4. 搜索功能测试

测试步骤：
  • 按 / 键打开搜索
  • 输入搜索词，如 "测试"
  • 按 n 查找下一个
  • 按 N 查找上一个
  • 按 Esc 退出搜索

5. 表单输入测试

测试步骤：
  • 点击输入框进入插入模式
  • 输入一些文本
  • 按 Esc 退出插入模式
  • 测试在表单中的 Vim 键盘操作

6. 滚动测试区域

这是一个长内容区域，用于测试滚动功能。使用以下键盘快捷键测试滚动：

  • j - 向下滚动
  • k - 向上滚动
  • d - 向下滚动半页
  • u - 向上滚动半页
  • Space - 向下滚动一页
  • Shift+Space - 向上滚动一页
